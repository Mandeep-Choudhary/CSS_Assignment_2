CSS Units



em

In em font size of the parent is taken as base to derive new font size.

Text with size 14px

Text with size 2em=28px

Text with size 2em=56px





rem

In rem standard font size i.e., 16px is used to derive new font size.It is independent of parent font size

Text with size 14px

Text with size 2rem=32px

Text with size 2rem=32px





vw

1vw is equal to 1% of viewport width

5vw font





vh

1vh is equal to 1% of viewport height

5vh font





CSS Combinators

Child Selector >

It targets child inside the specified parent e.g 'div > p' will apply to all p child inside parent div

Span inside div targeted using 'div > span' and background color set to skyblue




Adjacent Sibling selector +

It targets immediate sibling after specified tag.e.g 'div + p' will apply to immediate sibling p after div.

Div

Immediate p after div.Background color set to yellow using div + p

Not immediate p after div.No effect on background color





General Sibling Selector ~

It targets all sibling after specified tag.e.g 'div ~ p' will apply to all sibling p after div.

Div


A b tag after div.Background color set to light green using div ~ b

Anothet b tag after div
